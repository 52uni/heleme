# WaterReminder / 喝水提醒

一款 Android 喝水提醒应用，通过定时提醒、饮水记录和达标追踪，帮助你养成健康饮水习惯。

---

## 功能

- **水波进度球** — 自定义动画进度展示今日饮水量 vs 每日目标
- **快捷打卡** — 一键记录预设量（150/200/250/500ml）或自定义输入
- **三种提醒机制**
  - 间隔提醒（基于人群类型配置）
  - 小周期催促（未打卡时每 5 分钟再次提醒）
  - 固定时间提醒（可自定义标签）
- **全屏锁屏提醒** — 闹钟触发时亮屏显示喝水提醒，支持锁屏界面
- **桌面小组件** — 展示今日进度，支持一键喝水
- **日/月饮水记录** — 柱状图 + 折线图可视化
- **人群类型** — 预置成年男性/成年女性/青少年/运动达人 + 自定义类型
- **连续达标天数** — 追踪每日目标完成情况
- **厂商适配** — 电池优化白名单 + 各厂商自启动引导
