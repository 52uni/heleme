# WaterReminder

喝水提醒助手 - 一个简洁的 Android 喝水打卡应用。

## 下载

最新 APK 请前往 [Releases](https://github.com/52uni/heleme/releases) 页面下载。

## 功能

- 定时喝水提醒，灵活设置每日目标
- 固定时间 + 间隔循环双模式提醒
- 通知栏快捷打卡，无需解锁手机
- 桌面小组件，随时查看进度
- 多种人群类型，适配不同饮水需求
- 喝水记录与统计洞察
- 数据导出 CSV
- 隐私说明页面，完全本地存储
